## 🧭 主な機能 (Core Features)

- **✨ 三面大盤の自動解析**: 生年月日から「本命星（本質）」「月命星（精神）」「日命星（行動）」および気学上の出生基準年月を完全自動算出。
- **🔮 2026年（五黄中宮）対応・高精度吉凶判定**: 年盤と月盤が重なる奇跡のトラップ（2026年6月度など）においても、年盤の歳破（南西）と月盤の月破（北）を完全に独立して判定する超精密ロジック。
- **🗓️ 未来シミュレーター**: 月盤を切り替えることで、数ヶ月先の旅行や移動の計画を直感的に練ることが可能。
